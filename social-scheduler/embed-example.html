Example 2: Inline Form

How to Use

To embed a form on your website:

  1. Go to your Social Scheduler dashboard
  2. Navigate to Forms Management
  3. Click the "Embed" button for the form you want to embed
  4. Copy the generated embed code
  5. Paste it into your website's HTML

Embed Code Example:

<!-- Social Scheduler Form Embed --> <div id="social-scheduler-form-123" style="min-width: 320px; height: 600px;"></div> <script type="text/javascript" src="https://your-domain.com/embed.js" async></script> <script type="application/social-scheduler-embed"> { "type": "iframe", "formId": "form-123", "containerId": "social-scheduler-form-123", "width": "100%", "height": "600px", "theme": "light", "hideHeader": false } </script> <!-- Social Scheduler Form Embed End -->
